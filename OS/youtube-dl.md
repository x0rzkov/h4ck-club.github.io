###### youtube downloder
```bash
youtube-dl -U # update
youtube-dl <video_url> # download video
youtube-dl -F <video_url> # list of available formats
youtube-dl -f 37 <video_url> # download file no 37
youtube-dl --list-subs <video_url> # list all subs
youtube-dl -x <video_url> # download audio only
youtube-dl -x --audio-format mp3 <video_url> # download audio
youtube-dlg # GUI
youtube-dl -a youtube_links.txt # download from a file list 
```
